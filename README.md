# Revenue & Profit Analysis Dashboard
## Overview
This project presents a data-driven dashboard analyzing revenue, cost, and profit across products, customers, and stores. It aims to generate actionable insights to support business decision-making.

## Objectives
- Analyze revenue and profit trends over time  
- Identify top-performing products and stores  
- Evaluate the impact of discounts on profitability  
- Understand customer purchasing patterns  

## Dataset
The dataset includes:
- Order ID and Order Date  
- Product, Store, and Customer IDs  
- Quantity, Unit Price, and Discount  
- Revenue, Cost, and Profit  

## Tools Used
- Excel / R  and Data Studio for Dashboard(choose what you used)

## Key Insights
- Revenue trends vary across time and locations  
- Certain products and stores drive most profits  
- Discounts influence profit margins significantly
